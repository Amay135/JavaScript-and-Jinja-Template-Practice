# JavaScript and Jinja Template Practice

A simple practice project to learn and experiment with JavaScript and Jinja templating for web development.

## Features

- Basic JavaScript implementation
- Dynamic HTML rendering using Jinja templates
- Template variables and expressions
- Conditional statements and loops in Jinja
- Simple and beginner-friendly project structure

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Jinja2

## Getting Started

### Clone Repository

```bash
git clone https://github.com/Amay135/latihan-Js-Jinja.git
cd latihan-Js-Jinja
```

### Create Virtual Environment

```bash
python -m venv venv
```

### Activate Virtual Environment

Windows:

```bash
venv\Scripts\activate
```

Linux/macOS:

```bash
source venv/bin/activate
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run the Application

```bash
python app.py
```

Open your browser and visit:

```text
http://127.0.0.1:5000
```

## Learning Objectives

- Understand how Jinja templates work.
- Learn how to combine JavaScript with server-side rendering.
- Practice building dynamic web pages.
- Improve understanding of template inheritance and rendering.
